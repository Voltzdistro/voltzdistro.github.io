Frequently Asked Questions

Questions about the brand, tools, security, and philosophy.

Ask Less. Read More.

Here are the questions people ask — or should ask — before using anything I build. If your question isn’t here, you’re probably overthinking it.

Do I need to know code to use your tools?

Not at all. Just head to the Downloads page. Everything there is simplified for regular users.

Can I trust these files?

I don’t upload junk. No ads. No spyware. No hidden installers. Still, use common sense — run a scan if you're paranoid.

It didn’t work on my device. What now?

Read any instructions I included. If it's still broken, send me the details through the Request page or GitHub Issues.

Can I request a tool or app?

Yes. Use the Request Page. Be clear. Be specific. Don’t waste time.

Are these tools free?

Everything I share here is free to use unless stated otherwise. Respect the work. Don’t resell it.

Can I modify your code?

If it's in the Repo, you’re free to clone, edit, or fork it. Just follow the license and don’t claim it as your own.

Where’s the documentation?

Inside the files. README or instructions usually included. If it’s not there, it’s either obvious — or not ready.

Can I shout you out / credit you?

Sure. Not required, but always appreciated.

What systems do your tools support?

Mostly Windows. Some scripts might work on Linux too. If it doesn’t say, assume Windows.

Do you build mobile apps?

Rarely. Most of my tools are PC-first. If a mobile version exists, it’ll be mentioned in the description.

What kind of requests do you ignore?

Anything vague, bloated, or legally sketchy. I don’t build hacks, cheats, or spyware. Don’t ask.

Do you collect user data?

No trackers. No analytics. I build tools, not traps. What you run is yours.

Can I contribute to Voltzdistro?

If you’ve got skills, sure. Hit me up through GitHub or email. No hand-holding, though — come prepared.

Will this project disappear someday?

No. This isn't a hype project. It’s a mission. Even if I go quiet, I’m still building.
